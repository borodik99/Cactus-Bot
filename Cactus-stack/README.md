# 🌵 Cactus Watering Bot

Telegram‑бот на Node.js и [grammY](https://grammy.dev), который помогает по очереди поливать кактус **«Макс»**.  
Бот ведёт очередь, логирует поливы в PostgreSQL и поддерживает админ‑панель для управления доступом пользователей.

## Возможности

- Регистрация пользователей через `/start`
- Очередь на полив:
  - Встать в очередь / выйти
  - Посмотреть текущую очередь
  - Узнать, кто поливает сейчас и кто следующий
- Лог поливов (кто когда поливал и сколько)
- Напоминания о поливе по cron (каждую вторую среду в 10:00)
- Система доступа:
  - Заявки новых пользователей
  - Одобрение/отзыв доступа админом
  - Просмотр списка пользователей

## Технологии

- Node.js
- [grammY](https://grammy.dev) — Telegram Bot API
- PostgreSQL
- `node-cron`
- `dotenv`
